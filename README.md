## 介绍

可以自定义的虚拟宠物

## 功能

- [x] 支持导入png与gif格式的虚拟宠物

- [x] 支持GIF、PNG图片进行缩放操作

## 使用

安装依赖

```bash
pip install -r requirements.txt 
```

如果下载慢，中国大陆用户可以使用清华源进行依赖的下载:

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/
```

运行run.py:

```bash
python run.py
```
